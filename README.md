# SQL-Advertising-System-Failures-Report

## Sistema Publicitario - Base de Datos y Consulta SQL

Este proyecto presenta la estructura de una base de datos para un sistema publicitario, junto con un ejemplo de datos de clientes, campañas y eventos. 
También incluye una consulta SQL para obtener información sobre los clientes que han experimentado un número máximo de eventos de falla en sus campañas.
